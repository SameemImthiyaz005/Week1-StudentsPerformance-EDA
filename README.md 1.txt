# Week 1 - Student Performance Analysis

This project is part of the NSP Nexus Internship.

## 🔍 Project Description
We analyzed student exam scores using Python and visualized how gender, lunch type, parental education, and test preparation affect performance.

## 📁 Dataset Used
- Kaggle Dataset: Students Performance in Exams

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## 📊 Key Insights
- Students who completed the test preparation course scored higher.
- Females scored better in reading and writing.
- Standard lunch students performed better in all subjects.

## 📌 How to Run the Project
1. Open the `Week1-StudentPerformance-EDA.ipynb` notebook in Google Colab or Jupyter.
2. Run all cells step by step.
